| 類型  | 等效傳遞函數                                     |
| --- | ------------------------------------------ |
| 串聯  | $$G_{eq}(s) = G_1(s) \cdot G_2(s)$$        |
| 並聯  | $$G_{eq}(s) = G_1(s) + G_2(s)$$            |
| 負反饋 | $$G_{eq}(s) = \frac{G(s)}{1 + G(s) H(s)}$$ |
