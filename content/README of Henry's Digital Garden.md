歡迎來到我的數位大腦備份區！這裡是我在網路上的一方小天地，記錄了我的生活軌跡、學習筆記以及各種思考的過程。

## 📖 關於這裡

這個網站不是一個經過完美排版、正經八百的「部落格」，而是一個持續生長中的「數位花園 (Digital Garden)」。筆記之間互相連結，有些文章可能還在草稿階段，有些則是經過深思熟慮的總結。

你可能會在這裡探索到以下內容：
- **💻 數位生活與工具**：關於 Obsidian、自動化工作流、軟體使用心得。
- **🧠 心理學與情緒**：自我覺察、與焦慮共舞的心理學筆記。
- **🔬 科學與學習**：AI 人工智慧、流體力學、自動控制等硬核知識。
- **🎮 娛樂與日常**：遊戲日記、攝影紀錄 (Capture) 以及生活隨筆。

> **💡 導覽小建議**：
> 歡迎隨意點擊文章中的「內部連結」，或是透過左側的「資料夾樹狀圖」與右側的「關聯圖 (Graph View)」來一場知識的隨機漫遊。

## 🛠️ 技術架構 (How it's built)

這個網站的寫作與發布流程，全靠優秀的開源工具與自動化技術支撐：

- **大腦核心**：[Obsidian](https://obsidian.md/) (所有的筆記都在此誕生)
- **網站生成引擎**：[Quartz 4](https://quartz.jzhao.xyz/) (負責將 Markdown 轉換為漂亮的靜態網頁)
- **自動化部署**：GitHub Actions & GitHub Pages (一鍵 Push，全自動發布)
